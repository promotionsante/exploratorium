Description
Le projet vise à améliorer les interfaces entre le réseau des addictions et le monde hospitalier. Trois projets pilotes seront conduits dans trois hôpitaux distincts, un par région linguistique. Le Consortium va rechercher des profils d'hôpitaux différents, pour améliorer la transférabilité des enseignements du projet dans tous les hôpitaux.
